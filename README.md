# Slack Themes

## Usage

Copy the hex values, and go to `Preferences` > `Sidebar Theme` > `Custom Theme`.

Need help? See [Slack's instructions](https://get.slack.help/hc/en-us/articles/205166337-Customize-your-Slack-theme).
